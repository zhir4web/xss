About This Project

Hello! My name is Zhir, and this blog is part of my college project to learn and demonstrate web development using modern tools like Django and Tailwind CSS.

On the backend, it uses Django, a powerful Python web framework. On the frontend, it's styled with Tailwind CSS to make it look modern and responsive.

You can explore more by visiting the Home page or browsing through the Posts.
